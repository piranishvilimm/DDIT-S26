# Digital Disruption, Innovation, and Transformation

Instructor: Zeshan Ahmad  
Kutaisi International University, Finance and Information Management  
Master's Level Seminar

This repository is the working environment for the course. It is designed for a 12-student seminar organized into 3 teams of 4. It supports class sessions, inter-weekend sprint work, team research, and project artifact development across the semester.

## Course purpose

The course examines how digital technologies reshape industries, business models, operating logic, and competitive dynamics. It combines strategic management, digital innovation theory, case analysis, and consulting methods.

## What students will do here

- Access core frameworks and case materials
- Use structured worksheets during Weekend 1
- Save team outputs in a versioned workspace
- Build the foundation for the midterm and final project
- Track hypotheses, evidence, and strategic reasoning over time

## Weekend structure

- Weekend 1, March 14-15, 2026: foundations of digital disruption and consulting methods
- Weekend 2, April 4-5, 2026: AI strategy, platform competition, and strategic options
- Weekend 3, May 23-24, 2026: midterm presentations and transformation roadmap design
- Weekend 4, June 6-7, 2026: responsible AI, governance, and final integration

## Repository logic

The repository is organized around five layers:

1. Course-wide reference material
2. Framework summaries
3. Case materials
4. Weekend modules
5. Team project workspaces

## Rules for use

- Store all team work in your assigned team directory
- Name files clearly and consistently
- Keep claims tied to evidence and cite sources
- Use the provided templates before inventing your own formats
- Do not add side-topic artifacts that are outside the syllabus

## Team directories

Each team has a dedicated directory in:

`projects/team_directories/`

Use it to store meeting notes, issue trees, hypothesis logs, research notes, and deliverables.
