# Weekend 1 Module

## Dates
Saturday March 14, 2026  
Sunday March 15, 2026  
10:00 to 18:00 both days

## Theme
Foundations of digital disruption and consulting methods

## Learning objectives

By the end of Weekend 1, students should be able to:
- explain Rogers' five domains,
- distinguish disruption from sustaining innovation,
- apply Christensen's framework to real cases,
- explain why digital innovation behaves differently using Yoo et al.,
- use issue trees and hypothesis logs,
- analyze a client industry using competitive landscape methods,
- leave with a clear inter-weekend sprint plan.
