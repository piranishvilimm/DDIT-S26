# Value Chain Analysis

## Purpose

Value chain analysis breaks an industry into the stages where value is created, captured, or eroded. This helps identify where digital disruption lands first.

## Typical logic

A value chain may include:
- sourcing,
- production,
- distribution,
- sales,
- service,
- data feedback,
- financing,
- coordination layers.

## Questions to ask

- Which stages are being digitized?
- Which stages are being disintermediated?
- Which stages are gaining strategic importance?
- Who is capturing profit?
- Where do platforms insert themselves?
